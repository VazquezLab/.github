This is the [Vazquez Lab](https://lab.vazquez.bio) GitHub Repo! In addition to hosting repos associated with lab projects, this org page also hosts Wikis and FAQs for the Vazquez Lab, plus other resources.

If you're just joining the lab - WELCOME!!! You can find our [Getting Started](https://github.com/VazquezLab/Cheatsheets/blob/main/GettingStarted.md) guide in the [Cheat Sheets](https://github.com/vazquezlab/Cheatsheets) repo. 

Once you're onboarded, please make sure to create your own repo to keep track of your projects.
